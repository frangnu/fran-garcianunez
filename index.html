¿Qué hago?

En este apartado aparecen algunos de los proyectos (a modo de resumen). Los proyectos completos los podéis ver en mi GitHub personal.

Machine learning

A través de un Dataset de Kaggle, realicé un EDA sobre unos datos concretos de Strava. Creando un modelo de regresión logística para que a través de unas features, haga una predicción del segmento.

Reto de tripulaciones

Trabajo conjunto con el resto de los bootcamps, los cuales eran UX, Ciberseguridad y full stack. Ccreamos un producto para la Fundación Luzón, cuyo objetivo era el de conseguir donacionaciones donaciones para la misma. Desarrollamos una APP de donaciones programadas y no programadas a lo largo del mes.

Deep learning

Creación de una red neuronal. El proyecto se basó en que esa red tratara de distinguir, a través de fotografías, el tipo de bicicleta que le estábamos pasando. El modelo diferenciaba entre bicicleta de montaña y de carretera. Obteniendo un accuracy de un 73%.

Hundir la flota

Programación del juego hundir la flota con Python. Utilizando un poco de la programación orientada a objetos con el lenguaje indicado.
